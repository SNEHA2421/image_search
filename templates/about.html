Image Search Engine

Image Search Engine consists of retrieving the most visually similar images to a given query image from a database of images.

Float16 to save CPU/GPU memory based on apex, ResNet/DenseNet & Pytorch

- DataSet: CIFAR10, ResNet50, 1w Pictures collected by Web Crawler without label

- NetWork: Simple CNN(trained in CIFAR10), ResNet50(pretrained in ImageNet)
